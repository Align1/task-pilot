## 🎨 Creating Visual Pitch Decks

These markdown files are great for reference, but for actual presentations you'll want:

### **Tools to Use:**
1. **Pitch** (pitch.com) - Free, beautiful templates
2. **Canva** - Easy drag-and-drop
3. **Google Slides** - Simple, shareable
4. **PowerPoint** - Professional standard
5. **Keynote** - Mac users, very polished

### **Convert Markdown to Slides:**

**Option 1: Manual (Recommended)**
- Copy sections from markdown
- Paste into slides
- Add visuals, charts, screenshots
- ~2-3 hours of work

**Option 2: Use a Tool**
- Marp (markdown to slides)
- Slidev (developer-focused)
- Deckset (Mac only)

### **Slide Deck Structure (from INVESTOR_PITCH.md):**

```
1. Cover (Logo, tagline, your name)
2. Problem (3 bullets)
3. Solution (product screenshot)
4. Why Now (market timing)
5. Market Size (TAM/SAM/SOM)
6. Business Model (pricing tiers)
7. Traction (key metrics)
8. Competition (comparison table)
9. Competitive Advantage (4 key points)
10. Go-to-Market (3 phases)
11. Financial Projections (5-year table)
12. Team (photos + bios)
13. The Ask (amount, use of funds)
14. Vision (where we're going)
15. Contact (thank you + CTA)
```

**Tip:** Keep it to 15-20 slides max. Investors can read the detailed markdown docs later.

## 🎤 Pitch Practice Guide

### **Before Any Pitch:**

**1 Week Before:**
- [ ] Customize the relevant document
- [ ] Create visual slides if needed
- [ ] Prepare demo (video or live)
- [ ] Research the investor/sponsor
- [ ] Prepare 5 questions to ask them

**1 Day Before:**
- [ ] Print cheatsheet
- [ ] Test demo (internet + offline mode)
- [ ] Charge all devices
- [ ] Review objection handlers
- [ ] Get good sleep

**1 Hour Before:**
- [ ] Review pitch cheatsheet
- [ ] Practice 30-second pitch out loud
- [ ] Check appearance
- [ ] Arrive early (or log in early if virtual)
- [ ] Deep breaths, stay confident

## 📈 Tracking Your Pitches

**Create a Simple Spreadsheet:**

| Date | Contact | Type | Document Used | Outcome | Follow-Up | Status |
|------|---------|------|---------------|---------|-----------|--------|
| 11/15 | Jane Smith | Investor | Executive Summary | Interested | Send full deck | Pending |
| 11/18 | Tech Corp | Sponsor | Partnership Pitch | Meeting scheduled | 11/25 call | Active |

**Track Metrics:**
- Pitches sent
- Meetings secured (conversion rate)
- Second meetings (interest rate)
- Term sheets received
- Deals closed

**Goal:** ~5-10% of cold emails → meetings, ~10-20% of meetings → deals
